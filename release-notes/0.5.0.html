Vacay

Release Notes

Version 0.5.0

Tuesday, March 5, 2015

[NEW] Adds an option to keep read articles for less time.

[CHANGE] Max Sync Count is now ignored, Unread will always sync all unread articles. (Previously it was syncing the oldest X articles).

[CHANGE] Font sizes will no longer sync. They can now be set independently on the iPhone and iPad versions.

[FIX] Rotation bug when image viewed full screen.

[FIX] NYT feeds no longer show blank images.

[FIX] Facebook sharing should now work properly.

[FIX] The badge counter on the app icon would not be updated in the background.

[FIX] Crash when sharing with Mail

[FIX] Crash that could occur after clearing the image cache.

[FIX] Crash when bringing up Workflow or other remote view controllers.

[FIX] Settings could get reset to default.

[FIX] Visual issue with articles that include "tooltip" class elements.

[FIX] If a Fever user had a lot of unread items in Sparks, new articles weren't coming through.

If you have any problems at all, get in touch. If you enjoy Vacay, please consider leaving us a nice review.
